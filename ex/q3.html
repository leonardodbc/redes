Questão 3

Quais são os registros de DNS disponíveis para o endereço escolhido? Detalhe cada um dos registros encontrados.

- Registros DNS:
- wikipedia.org.
- ns2.wikimedia.org.
- ns1.wikimedia.org.
- ns0.wikimedia.org.

- Detalhamento:

- wikipedia.org é o endereço DNS correspondente ao IP 208.80.154.224 e é um registro A (address)

- ns2.wikimedia.org é o endereço DNS correspondente ao IP 91.198.174.239 e é tanto um registro A (address) quanto um registro NS (name server)

- ns1.wikimedia.org é o endereço DNS correspondente ao IP 208.80.153.231 e é tanto um registro A (address) quanto um registro NS (name server)

- ns0.wikimedia.org é o endereço DNS correspondente ao IP 208.80.154.238 e é tanto um registro A (address) quanto um registro NS (name server)

- Registro A (address): associa um endereço IP a um domínio

- Registro NS (name server): indicam quais servidores atuam como serviço DNS do site.
